// about

Hi, I'm Sumit.

I'm a full-stack engineer with 8+ years across the stack — Node, React and React Native, Postgres, and Kubernetes — currently a Technical Lead building SaaS products. Along the way: an MTech from BITS Pilani, four TopCoder challenge wins, and an open-source start with react-native-use-websocket, a WebSocket hook still in production use today.

These days I work differently: I direct AI agents rather than typing the code myself, and I build the missing infrastructure that makes that trustworthy — SREForge to test whether agents can genuinely resolve incidents, PrismaLens to let them investigate real ones, and mage-memory to give them durable knowledge. Each ships with proper documentation on its own domain.

Want to talk tooling, agents, or SRE? Email me, find me on GitHub, or connect on LinkedIn.
